# Natalie Cornejo – Project Portfolio

Hi, I'm Natalie Cornejo, a 4th year Data Science student at UC Berkeley with a focus on data analysis, economics, and real-world problem solving.

This repository is a collection of my projects in data science, analytics, and technology. It is designed to showcase my technical skills, thought process, and ability to use data to solve meaningful problems.


## 2022 U.S. Primary Election Analysis

This project analyzes the 2022 U.S. primary elections using FiveThirtyEight data to study what factors influence candidate success. I used causal inference and hypothesis testing to examine the effect of Trump endorsements and compare win rates across gender, race, and incumbency. This project highlights my experience with data cleaning, statistical analysis, and drawing insights from real-world political data.


## Beauty, Race, and Wages Analysis

This project investigates how physical attractiveness affects wages for female workers and whether this relationship differs by race. Using regression models and interaction terms on the Wooldridge “beauty” dataset, I analyzed how attractiveness, education, and other factors influence earnings.

The analysis reveals that returns to attractiveness vary by race, providing insight into labor market inequality. This project demonstrates my ability to apply econometric methods and interpret real-world data to uncover meaningful patterns.
